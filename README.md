# Traffic-IQ
🚦 Traffic IQ

Traffic IQ is an intelligent traffic analysis and simulation system designed to visualize, predict, and optimize urban traffic flow using modern web technologies and data-driven logic. It helps understand congestion patterns, signal timing efficiency, and vehicle movement behavior in a simplified smart-city model.

📌 Features
🚗 Real-time traffic simulation
📊 Traffic density visualization
🧠 Smart signal optimization logic (AI/Rule-based)
🛣️ Multi-road intersection handling
📍 Vehicle movement tracking
📉 Congestion heatmap display
⚡ Fast and lightweight frontend experience
📱 Responsive UI for desktop and mobile
